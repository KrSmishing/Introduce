# 📱 스미싱(Smishing) 탐지 AI를 위한 데이터셋 구축 및 벤치마크

## 1. 스미싱의 위험성

스미싱(Smishing)은 **SMS(Short Message Service)와 피싱(Phishing)의 합성어**로,  
문자 메시지를 통해 악성 링크 클릭이나 개인정보 입력을 유도하는 사이버 범죄입니다.

최근 스미싱 공격은 단순한 링크 유도 수준을 넘어,
- 택배 배송
- 공공기관 사칭
- 금융·인증 절차 위장

등 **일상적인 문구를 정교하게 모방**하며 빠르게 진화하고 있습니다.  
이로 인해 금전적 피해뿐만 아니라 개인정보 유출, 2차 범죄로까지 이어지는 심각한 사회적 문제가 되고 있습니다.

---

## 2. 스미싱 탐지를 위한 AI의 필요성

기존의 규칙 기반(rule-based) 또는 키워드 중심 필터링 방식은  
다음과 같은 한계를 가지고 있습니다.

- 문구 변형에 취약함
- 신종 스미싱 패턴 대응 지연
- 오탐(false positive) 증가

이에 따라 **문맥을 이해하고 의미 기반으로 판단할 수 있는 AI 모델**의 필요성이 커지고 있습니다.  
특히 자연어 처리(NLP) 기반 모델은 스미싱 문자의 의도와 맥락을 학습하여  
보다 정밀한 탐지가 가능하다는 장점이 있습니다.

## 3. 데이터셋 구축 개요

본 프로젝트에서는 **스미싱 탐지 AI 학습 및 평가를 위한 데이터셋**을 구축하였습니다.

### 📌 데이터셋 특징
- 스미싱 문자 / 정상 문자 이진 분류
- 실제 사례 기반 문구 구성
- 다양한 도메인(금융, 택배, 공공기관 등) 포함

데이터셋은 모델 학습뿐만 아니라 **공정한 성능 비교를 위한 벤치마크** 용도로 활용할 수 있도록 구성되었습니다.
